## 1. 前端概述
EZLinker前端项目是基于Antdesign构建，在此之上做了部分组件封装，因此如果需要二次开发，建议提前学习Antdesign-React相关知识。
## 2. 相关技术
1. react
2. typescript
3. ant-design-pro
4. react-hooks
5. swr
## 2. 项目构建
```shell
git clone https://github.com/ssloth/ezlinker-frontend.git
cd ezlinker-frontend
yarn
yarn start
```
